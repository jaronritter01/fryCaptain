Note: This Project is no longer in production and cannot be viewed

Setup:

1. Setup up a virtual env with python using: py -m venv envName
2. Activate the venv using: .\envName\Scripts\activate.bat
3. install requirements from requirements.txt: pip install -r requirements.txt
4. activate the testing server(have to be in the same directory as manage.py): py manage.py runserver
5. open project on local host
